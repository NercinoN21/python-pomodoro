## 🚀 Running the Project:

- ### Clone the repository:
```
git clone https://github.com/NercinoN21/python-pomodoro
```

- ### Enter the repository:
```
cd python-pomodoro
```

- ### Check the requirements:
```
pip install -r requirements.txt
```

- ### Run the APP:
```
python app/scripts/main.py
```
